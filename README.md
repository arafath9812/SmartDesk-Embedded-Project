# SmartDesk - Ergonomic Workspace Assistant

A wireless desk system built with two BBC micro:bits to promote healthier work habits.

## What it does
- Monitors user posture using an ultrasonic sensor
- Tracks sitting time and reminds user to take breaks
- OLED display shows real-time stats (posture, comfort score, timer)
- RGB LED responds to ambient light levels

## Tech Stack
- MicroPython
- 2.4GHz radio communication
- I2C OLED display
- Ultrasonic sensor
- PWM for RGB LED

## Files
- `node 1-3 with radio.py` – Main unit with sensors and radio transmitter
- `node 2_4 final code.py` – Receiver unit with OLED display and LED
